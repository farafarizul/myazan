Project Name : "myAzan"
Technolgy : Electronjs, windows 10 atau windows 11
Bahasa yang akan digunakan dalam system ini untuk paparan pengguna : Bahasa melayu

Objektif projek:
1. Membina satu app untuk windows.
2. App yang akan mengumandangkan waktu azan apabila tiba masanya mengikut zon pilihan.
3. App boleh dijalankan secara offline tanpa internet selepas data untuk zon dan tahun tersebut didownload dan save locally. Mungkin save di database atau save sebagai standalone fail, mana yang lebih baik.
4. app akan dijalan di windows 10 keatas dan menggunakan mini PC. jadi ia akan run 24jam.
5. app akan mengumandangkan azan dan notification melalui speaker apabila tiba waktunya.
6. app akan mengalunkan ayat-ayat alquran atau zikir semasa idle, iaitu semasa azan atau notification tidak dimainkan lagi.
7. Selepas azan atau notification siap play, app akan terus mengalunkan ayat-ayat alquran atau zikir seperti dalam setting.

Cadangan pelan pembangunan :
1. Mungkin boleh buat 3 mp3 player berasingan yang akan setiasa bersedia memainkan azan, notification dan zikir. Jadi tidak perlu berkongsi 1 player untuk semua.
2. sekiranya tidak berkongsi mp3 player, maka ketika azan play atau notification play, zikir boleh pause sebentar dan sambung semula selepas player-player lain stop playing.

Semasa app launch, check zon yang telah dipilih, seterusnya check adakah data untuk zon tersebut telah didownload locally?
- Seandainya belum ada, call API dan download dan save data pada tahun tersebut locally. Situasi ini berlaku seandainya berlaku perubahan tarikh dari satu tahun kepada tahun lain.

Ada page untuk ubah setting. dalam page tu dirangka untuk mempunyai setting seperti berikut :
1. Pilih ZON anda 
- User boleh memilih zon yang diingini mereka. Setiap kali zon dipilih, sistem akan check adakah data untuk zon tersebut pada tahun tersebut telah didownload? seandainya belum, maka download dan save data untuk zon tersebut dan tahun tersebut.
- Semua data yang didownload perlu disimpan secara berasingan, supaya akan datang sekiranya user mahu tukar-tukar zon, tidak perlu download semula untuk tahun tersebut.
2. MP3 files to play when system trigger adhan time.
- app akan setiasa check untuk trigger waktu azan.
- sekiranya telah tiba waktu azan megikut zon yang dipilih, app akan memainkan azan.
- ada setting untuk pilih mp3 fail yang akan dimainkan setibanya waktu azan.
- perlu ada 2 jenis mp3 waktu azan, iaitu azan subuh dan azan lain (zohor, asar, maghrib, isyak)
3. Notification {x} minutes before azan is playing.
- app akan check adakah waktu azan hampir tiba. seandainya sistem set notification adalah {x} minit, maka sistem akan play mp3 notification sebelum azan berkumandang mengikut {x} minit.
- perlu ada setting untuk pilih mp3 fail untuk setiap waktu azan (imsak,fajr,syuruk,dhuha,dhuhr,asr,maghrib,isha). Setiap waktu memerlukan setting berbeza jadi user boleh pilih notification berbeza untuk setiap waktu azan termasuk imsak,syuruk dan dhuha
- Notification untuk setiap waktu (imsak,fajr,syuruk,dhuha,dhuhr,asr,maghrib,isha) boleh di enable atau disable.
4. User boleh pilih folder yang terisi dengan pelbagai mp3 ayat-ayat alquran.
- perlu ada setting untuk user pilih folder yang mengandungi pelbagai mp3 ayat-ayat alquran.
- system akan memainkan mp3 dalam folder ini secara continuous mengikut susunan filename. mp3 dengan filename 01- akan dimainkan paling awal. bermakna sorting by name.
- Seandainya telah habis play semua fail, ulang semula dari filename 01-
- Setiap kali notification hendak mula atau azan hendak mula, stop play fail-fail ini. berhenti putar ayat-ayat alquran untuk memberi laluan kepada mp3 notification atau mp3 azan.
- Ada setting untuk enable play ayat-ayat alquran atau disable.
